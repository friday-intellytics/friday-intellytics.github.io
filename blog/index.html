Friday Thesis

Our emerging belief system

Subject to change. Four short thesis we find today that frame the consulting work and the way we think modern data teams should operate.

Thesis 1

Self-Serve Analytics is a Myth

Every new technology right from Excel to no code BI tools made the same promise. But never delivered. It rather added to the bottleneck in Analytics workflow. Same is happening with A.I.
Thesis 2

AI Should Assist, Not Replace

The right system removes grunt work but keeps the judgment with the data practitioner. Same with A.I. systems which are capable of removing even more of the grunt work.
Thesis 3

Data Context. Not Decision Context

A correct number is not the same thing as a useful recommendation. Analytics exists to bridge that gap.
Thesis 4

The Analytics Experience Needs a Rebuild

Jupyter notebooks is the last best thing that ever happened to Analytics Experience Everything else is borrowed from software development. Analytics need more that a TUI. It needs a dedicated workspace supercharged with A.I.
