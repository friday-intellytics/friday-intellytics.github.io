Core Thesis

The four ideas shaping our point of view, how we think about A.I. and build for data teams

01

Self-Serve Analytics is a Myth

Every new technology right from Excel to no code BI tools made the same promise. But never delivered. It rather added to the bottleneck in Analytics workflow. Same is happening with A.I. Trust me no exec is going to build their own dashboard

02

AI Should Assist, Not Replace

The right system removes grunt work but keeps the judgment with the data practitioner. Same with A.I. systems which are capable of removing even more of the grunt work.

03

Context is the key

A correct number is not the same thing as an useful recommendation. Analytics exists to bridge that gap. And the context required for generating that insight is very different. A.I. must have the data context and decision context to come up with useful insight.

04

The Analytics Experience Needs a Rebuild

Jupyter notebooks is the last best thing that ever happened to Analytics Experience. Everything else is borrowed from software development. Analytics need more that a TUI. It needs a dedicated workspace supercharged with A.I. and an intelligent assistant that remembers.
